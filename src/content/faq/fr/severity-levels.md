---
question: Que signifient les niveaux de sévérité ?
category: Utilisation de Zonemaster
---

Info
: Le message indique une information pouvant être utile pour la personne administrant la zone, mais qui n’est en aucun cas une anomalie.

Note
: Le message indique un élément dont la personne administrant la zone devrait prendre connaissance sans qu’il ne s’agisse nécessairement d’un problème de configuration.

Avertissement
: Le message indique un élément pouvant causer un dysfonctionnement dans certaines circonstances, mais qu’une personne non avertie a peu de chances de repérer.

Erreur
: Le message indique une anomalie pouvant provoquer de façon (quasi) certaine un dysfonctionnement de la zone testée, sans être suffisamment grave pour rendre l’intégralité de la zone inaccessible.

Critique
: Le message indique une erreur très grave.
